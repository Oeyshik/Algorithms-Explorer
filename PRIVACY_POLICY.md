# Privacy Policy for Algorithms Explorer

**Last updated:** April 09, 2026

## Overview

Algorithms Explorer is a browser extension that helps users quickly look up the time and space complexities of common algorithms. This privacy policy explains how the extension handles user data.

## Data Collection

**Algorithms Explorer does not collect, store, or transmit any personal data.**

There are no user accounts, no login systems, and no forms that gather personal information.

## Data Storage

All algorithm data (names, time complexities, space complexities, and descriptions) is stored **locally within the extension** itself. No data is sent to any external server or third party.

## Permissions

This extension does not request any sensitive browser permissions. It does not access your browsing history, tabs, cookies, or any website content.

## Third-Party Services

This extension does not use any third-party analytics, advertising networks, or tracking services.

## Changes to This Policy

If this privacy policy is updated in the future, the changes will be reflected on this page with a revised "Last updated" date.

## Contact

If you have any questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/Oeyshik/Algorithms-Explorer).
